# Test Search Git Hub

Приложение сделано в рамках тестового задания

## Resources

- [Api Git Hub](https://docs.github.com/en/rest)
- [Техническое задание](https://docs.google.com/document/d/1LeAR1KQQS2lxqmBg8JHriz9R84JCn3CB58q4qjq5_lE/edit)
- [Скачать APK](https://gitlab.com/Asl2000/git_project_test/-/tree/main/assets/apk/app-release.apk)

## Stack

- DI (GetIt)
- State manager (Redux, Provider)
- Rest api (Dio)
- DTO (Freezed)
- Storage (Shared Preference)

## Функции

- Смена темы (Светлая, Темная)
- Смена локализвации (Русский, Английский)
- Поиск пользователей по Git Hub
- Просмотр страницы пользователя
- Просмотр списка репозиториев пользователя
- Просмотр списка подписчиков пользователя
